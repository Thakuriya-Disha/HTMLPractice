II. Classes and IDs:

  • Both class and IDs are selectors, they are used to give name to elements. Following are their differences:
  • Classes are not unique. IDs are unique.
  • There can be multiple classes. Where as there should be just one ID with the same value in a file.
  • Classes are denoted with dot. IDs are denoted with hash.

V. Block Elements vs Inline Elements

Block: Their attributes such as height, width, size can be changed using CSS.
Inline: Their attributes such as height, width, size cannot be changed.

Block: They take up the whole width of the screen.
Inline: They take only the required space.

Block: Always starts from a new line.
Inline: Not necessary to start from a new line.

Block: Their examples are <div>, <h1>, <p>
Inline: Their examples are <span>, <em>, <strong>

VI. <em> Tag vs <strong> Tag

The <em> & <strong> tags are preferred over <i> & <b> tags as they increase webpage's SEO Rankings.
strong tag is used to show the importance of a text by making it bold.
em tag is used to put emphasis on a text by italicizing it.

VII. What are Attributes in HTML?

Attributes give additional information about a tag. They enhance the properties of the tag.
They should always be written in lowercase alphabets. They must always be given in the opening tag.
Multiple attributes can be used in a particular tag and must be separated by space.

VIII. HTML Tag vs HTML Element

Tag consists of the opening and closing tags, or self-closing tags.
If we give content inside the tag, it becomes an element.
Example of only tag: <h1></h1>       Example of an element: <h1> Content </h1>
Tag + Content = Element
